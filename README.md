# Python Socket Chat (Server & Client)

This project demonstrates a simple chat system using Python socket programming
over TCP on localhost port 9999.

## Requirements
- Python 3.x

## Files
- server.py
- client.py

## How to Run

### Terminal 1 (Server):
python server.py

### Terminal 2 (Client):
python client.py

Then type messages on either side to chat.

## Stop
Press Ctrl + C

## Note
Supports only one client connection.
